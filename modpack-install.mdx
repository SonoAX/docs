## Launcher selection

While it is possible to install AXQOL on many Minecraft launchers, for the sake of keeping this short this guide focuses on installing it on the [Modrinth App](https://modrinth.com/app).

## Modrinth App Installation

1. Before starting please be sure to have the Modrinth App installed and updated to the latest version
2. Once you've successfully installed it and logged into your Minecraft account, go to "**Discover content**" (the compass icon in the top left section)
3. Here, make sure to be in the "**Modpacks**" section, and search "AX's QOL Pack".
4. Open the modpack's page by clicking it
5. Choose your version, or just go with the latest by clicking the green "**Install**" button on the right
6. Wait for the resources to finish downloading
7. That's it! You can now go to the "**Library**" page (Bookshelf icon underneath "**Discover content**"), select the AXQOL instance and click "**Play**".
